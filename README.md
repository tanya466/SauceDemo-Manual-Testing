# SauceDemo Manual Testing Project

## Project Overview
This project demonstrates end-to-end manual testing of the SauceDemo web application. The project includes complete testing documentation prepared during the Software Testing Life Cycle (STLC).

## Application Tested
https://www.saucedemo.com/

## Project Deliverables
The Excel workbook contains the following testing artifacts:

- Test Plan
- Test Cases
- Requirement Traceability Matrix (RTM)
- Bug Report
- Test Summary Report

## Testing Scope
- Login Functionality

- Cart Management
- Checkout Process
